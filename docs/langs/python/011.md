# 11. break, continue

---

## 1. 키워드

- `break`
- `continue`

<br/>

## 2. `break`, `continue`로 반복문 제어하기

- `break`는 제어흐름을 중단하고 빠져나오지만, `continue`는 제어흐름을 유지한 상태에서 코드의 실행만 건너뛰는 역할을 한다.
- 다른 프로그래밍 언어와 차이점이 없으므로 넘어 간다.
