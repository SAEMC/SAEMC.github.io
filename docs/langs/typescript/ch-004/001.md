# 1. TS의 타입 개념

---

## 1. 타입스크립트에서 타입이란?

- 타입스크립트에서의 타입이라는 개념은 크게 다음 다섯 가지 범주를 의미한다.

<br/>

1] 타입 별칭: `type sn = number | string;`

2] 인터페이스: `interface I { x: number[]; }`

3] 클래스: `class C {}`

4] 이넘: `enum E { A, B, C }`

5] 타입을 가리키는 `import` 구문
