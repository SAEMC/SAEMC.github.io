# 2. DROP DATABASE

---

## 1. `DROP DATABASE` Statement

- `DROP DATABASE` 문은 기존 SQL 데이터베이스를 삭제하는 데 사용된다.

<br/>

## 2. Syntax

```sql
DROP DATABASE databasename;
```

<br/>

## 3. `DROP DATABASE` Example

- 다음은 기존 데이터베이스 `testDB`를 삭제한다.

<br/>

```sql
DROP DATABASE testDB;
```
