# 1. Git 써보기

---

## 1. Repository 생성하기

```shell
git init
```

<br/>

## 2. User 정보 설정하기

```shell
git config user.name "ExampleName"
git config user.email "example@example.com"
```

<br/>

## 3. Working Tree에서 Staging Area로 저장하기

```shell
git add ExampleFile
git add .
```

<br/>

## 4. Staging Area에서 Repository로 저장하기

```shell
git commit -m "ExampleMessage"
```

<br/>

## 5. 상태 확인하기

```shell
git status
```

<br/>

## 6. Git add 취소하기

```shell
git reset ExampleFile
```
